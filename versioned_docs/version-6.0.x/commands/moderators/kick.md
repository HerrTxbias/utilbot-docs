## Command
```txt
/mod kick <user>
```

## Description
Kicks a Member.

## Permission
`KICK_MEMBERS`

## Arguments
| Name | Type | Description | Sample Data |
| ---- | ---- | ----------- | ----------- |
| user | USER | A user (ID should also work) | @HerrTxbias |
