## Command
```
p!changePrefix <prefix> / p!cp <prefix>
```

## Description
Changes the Prefix of the bot.

## Permission
`ADMINISTRATOR`

## Arguments
| Name | Type | Description | Sample Data |
| ---- | ---- | ----------- | ----------- |
| prefix | String | The new Prefix | ! |
