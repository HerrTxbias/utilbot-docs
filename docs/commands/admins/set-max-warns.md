## Command
```
p!setMaxWarns <count> / p!smw <count>
```

## Description
Sets the maximum warn count.

## Permission
`ADMINISTRATOR`

## Arguments
| Name | Type | Description | Sample Data |
| ---- | ---- | ----------- | ----------- |
| count | Integer | The Amount of the maximum Warns. | 20 |
