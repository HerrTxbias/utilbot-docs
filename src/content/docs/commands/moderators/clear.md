## Command
```txt
/mod clear [message_num]
```

## Description
Deletes all, or a specific amount of messages in the Chat.

## Permission
`MANAGE_MESSAGES`

## Arguments
| Name | Type | Description | Sample Data |
| ---- | ---- | ----------- | ----------- |
| `message_num` | number | The amount of messages to delete. Argument is optional. | `20` |
